## 看题分析

```js
var a = 12;
var b = a;
b = 13;
console.log(a); // 12
```

![stack](img/stack_01.png)

```js
var a = { n: 12 };
var b = a;
b = {
  n: 13,
};
console.log(a.n); // 12
```

![stack](img/stack_02.jpg)

```js
var a = { n: 12 };
var b = a;
b["n"] = 13;
console.log(a.n); // 13
```

![stack](img/stack_03.jpg)

```js
var a = { n: 1 };
var b = a;
a.x = a = { n: 2 };
console.log(a.x); // undefined
console.log(b); // {n:1,x:{n:2}}
```

a.x = a = {n:2}

优先级：

1 首先会访问 a.x 优先级最高
2 a.x = {n:2}
3 a = a.x
